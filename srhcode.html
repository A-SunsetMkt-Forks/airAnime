搜索指令

这里是airAnime拓展功能「搜索指令」的说明文档。

搜索指令的存在，可以更加快速、精确地查询到您需要的番剧。

如要使用，请仔细阅读。

指定搜索源代码（一次搜索中只可使用其一）

1. only:\?/ -只搜索指定数据源

例子：'散华礼弥 only:\b\d/'（只在 哔哩哔哩和嘀哩嘀哩 数据源中搜索散华礼弥番剧）

2. exc:\?/ -只搜索除指定的数据源

例子：'物语系列 exc:\bda\i\p\l\f\y/'（只在 哔哩哔哩和嘀哩嘀哩 数据源中搜索物语系列）

此代码必须小写 位于标题后 ，并且用 空格 隔开。开始:'only:',结束:'/' 或 开始:'exc:',结束:'/'.

通用形式：title code

数据源代码：
bilibili(哔哩哔哩): \b
dilidili(嘀哩嘀哩): \d
Fcdm(风车动漫): \f
PPTV(聚力): \p
Letv(乐视): \l
iQiyi(爱奇艺): \i
Youku(优酷): \y
BaiduAll(百度集合搜索): \bda

哔哩哔哩数据源为必搜索源，所以以上搜索指令对其无效

指令集

Name Example
only:\?/ 只搜索指定数据源 散华礼弥 only:\b\d/
exc:\?/ 只搜索除指定的数据源 物语系列 exc:\bda\i\p\l\f\y/
